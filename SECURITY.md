# Security Policy

APIForge takes security seriously.

---

## 🚨 Reporting a Vulnerability

If you discover a security issue, **do NOT open a public issue**.

Please report it privately:

📧 zeeshamushtaqdev786@gmail.com

---

## 🔄 Response Timeline

- Acknowledgement: within 48 hours
- Investigation: as soon as possible
- Fix release: depending on severity

---

## 🛡 Severity Levels

- Critical: API key leaks, auth bypass
- High: data exposure
- Medium: partial vulnerability
- Low: minor issue

---

## 🙏 Responsible Disclosure

We appreciate responsible disclosure and may credit reporters if desired.
